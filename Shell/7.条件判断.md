## 按照文件类型判断

![image-20201006122239431](.\images\image-20201006122239431.png)

格式：test [测试选项] [文件路径]  或 [ 测试选项  文件路径 ]

```
test -e /root/install.log

[ -e /root/install.log ]
```

> 注意执行后不会返回结果，需要调用显示上一次语句结果命令才能显示
>
> 还要注意[]语句中的空格不能省

```
echo $?
```

上面很不方便可以通过&&来处理，类似三元表达式

```
[-e /root/install.log] && echo "yes" || echo "no"
```
